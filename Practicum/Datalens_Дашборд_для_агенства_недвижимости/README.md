[Дашборд для агенства недвижимости](https://datalens.yandex/jyoyxv8le7aq5)

### Предназначение дашборда:

Быстрее обрабатывать клиентские запросы, которые просят подобрать недвижимость в разных населенных пунктах Санкт-Петербурга и Ленинградской области.

Дашборд поможет ответить на вопросы:

* Как быстро продается недвижимость (найти топ городов по самой быстрой продаже),
* Сколько квартир предложено на рынке и какая их средняя цена (в том числе по регионам),
* Какая средняя цена одного квадратного метра в разных населенных пунктах,
* Какие у жилья тип, планировка, есть ли рядом парки и водоемы,
* Как размещают объявления в зависимости от времени года.
