## [Дашборд для сервиса доставки еды](https://datalens.yandex/3y5c590c8axso)

### Предназначение дашборда:

Визуализация ключевых метрик для проверки состояния бизнеса. Заказчиком выступает сервис по доставке еды из Саранска.

Дашборд поможет ответить на следующие вопросы:

* Увеличивается или снижается DAU?
* Наблюдаются ли аномалии в конверсии?
* Как меняется средний чек с течением времени?
* Как меняется Retention Rate в зависимости от периода?
* Какие блюда и рестораны входят в топ по LTV?
