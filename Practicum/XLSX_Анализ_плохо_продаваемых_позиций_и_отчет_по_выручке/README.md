### Предназначение отчета

* Определить, в каком регионе кофейни приносят больше всего прибыли,
* Составить топ прибыльных товаров,
* Посчитать выручку в разрезе каждого региона и товара,
* Проанализировать количество продаж товаров за разные периоды,
* Составить топ самых непопулярных товаров по кофейням каждого региона.
