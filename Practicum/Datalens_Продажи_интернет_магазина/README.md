## [Дашборд по продажам интернет-магазина](https://datalens.yandex/6ilusl36rgsws)

### Предназначение дашборда:

Визуализация бизнес-метрик для интернет-магазина.

Дашборд помогает получить информацию:
* Общее число пользователей и покупателей,
* Рассчет среднего чека и конверсии,
* Самые популярные товары по категориям,
* Подробные данные о заказах по категориям
