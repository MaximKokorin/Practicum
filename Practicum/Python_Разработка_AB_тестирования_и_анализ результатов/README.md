### Описание задачи 
Команда разработчиков, работающая над развлекательным приложением с функцией «бесконечной» ленты, создала новый алгоритм рекомендаций, который, по их мнению, будет показывать более интересный контент для каждого пользователя. Нам необходимо рассчитать параметры A/B-теста, который позволит проверить эту гипотезу, и проанализировать его результаты.
